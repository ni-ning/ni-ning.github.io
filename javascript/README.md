

## 变量

```
1. 必须使用字母、下划线(_)、$开始
2. 多个英文字母 驼峰 myName
3. 不用使用js中关键字和保留字来命名
4. 严格区分大小写

var myName = 'linda';
var $http = 'HTTP';
```

## 数据类型
```
基本数据类型
Number String Boolean undefined null

引用数据类型
Object Array Function


var num = -3.1415;
var isTrue = (typeof num === 'sting');  // true
alert(isTrue);

var x;
alert(x);	// undefined
alert(typeof x === 'undefined'); // true

// 空对象
var y = null;
alert(y);
alert(typeof y === 'object')	// true
```
